# Basic File Organization

Organização básica para arquivos HTML | CSS | JS | IMAGENS.
## Estrutura
```bash
    index.html
    |_assets
    |	|css/
    |	|   |style.css
    |	|img/
    |	|   |example-img.jpg
    |  	|js/
        |   |main.js
```
## Rodando localmente

Clone o projeto ou Baixe o ZIP

```bash
  git clone https://github.com/Peagah-Vieira/Basic-File-Organization.git
```
