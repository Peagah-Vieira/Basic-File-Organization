# Basic File Organization

Basic organization for HTML files | css | JS | IMAGES.
## Structure
```bash
    index.html
    |_assets
    |	|css/
    |	|   |style.css
    |	|img/
    |	|   |example-img.jpg
    |  	|js/
        |   |main.js
```
## Running Locally

Clone the project or Download the ZIP

```bash
  git clone https://github.com/Peagah-Vieira/Basic-File-Organization.git
```
